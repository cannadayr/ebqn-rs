Usage
-----

    {ok,R} = ebqn:init_r(),
    {ok,C} = ebqn:init_c(R),
    {ok,P} = ebqn:compile(R,C,<<"{𝕩+1}"/utf8>>),
    {ok,3.0} = ebqn:callp(P,2.0).
